# Real-Time Cryptocurrency Data Engineering Pipeline

## Project Overview
This project implements a real-time data engineering pipeline that collects cryptocurrency market data, processes it using ETL principles, and prepares it for analytics and visualization.

## Technologies Used
- Python
- REST API (CoinGecko)
- Pandas
- PostgreSQL (Upcoming)
- Streamlit (Upcoming)
